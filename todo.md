## 3. 阶段 P1：流式解析器与安全契约 (Parser Limits) - *[已完成]*
- [x] **8位 C1 控制序列解析**：支持 `0x80..=0x9F` 在 Ground/String 状态下的 C1 单字节解析（如 `0x9B` 作为 CSI 等），并可通过配置开关控制。
- [x] **CSI/DCS 所有中间字节识别**：支持 `0x20..=0x2F` 范围内的多个中间字节，并在超出 limit 时报错。
- [x] **冒号子参数 (Colon Sub-parameters) 支持**：支持 `:` 分隔符及 `38:2::R:G:B`、`38:5:idx` 等子参数格式，防止其报错忽略。
- [x] **私有标记防混淆**：独立存储并区隔 `<`, `=`, `>`, `?` 不同的私有序列起始标记。
- [x] **C0 控制字符就地执行**：在 CSI / DCS 参数积累过程中，遇到的 C0 字符（如 `CR`、`BS` 等）能立即执行而不会中断/抛弃当前序列。
- [x] **超长字符串安全截断**：超过限制后能安全截断并不限制分配内存，但保持扫描终止符（`ST` / `BEL`）。

## 4. 阶段 P2：终端状态模型与屏幕语义 (State & Screen Semantics)
- [ ] **挂起换行状态 (Pending Wrap)**：实现标准的 DECAWM（自动换行）逻辑。写满最后一列时，游标不应立即换行，而是保持挂起状态，在输入下一个字符时才执行换行。
- [ ] **游标可见性控制 (DECTCEM / ?25)**：在 `Screen` 状态中增加游标显示/隐藏标志，并处理 `?25 h/l` 序列。
- [ ] **原点模式 (DECOM / ?6)**：光标定位指令在 DECOM 开启时，坐标应限制并相对于垂直滚动区域，而非物理屏幕。
- [ ] **左右水平边距 (DECLRMM / ?69)**：实现水平左右边距的存储，并使定位、插入、删除、清屏等操作支持水平边距。
- [ ] **自定义制表位设置**：实现 `HTS` (设置当前列为制表位)、`TBC` (清除当前列/所有制表位)，恢复 RIS 重置制表位的功能。
- [ ] **字符插入/删除时的擦除属性填充**：在新产生的空白区域，使用当前的擦除背景色属性进行填充。
- [ ] **保护字符与选择性擦除 (DECSCA / DECSED / DECSEL)**：实现字符受保护状态，使其在选择性擦除指令下不被清空。
- [ ] **整洁的软复位 (DECSTR)**：实现 `CSI ! p` 软复位，不丢失滚动历史地重置部分终端状态。

## 5. 阶段 P3：核心 ESC/CSI/DEC 兼容片 (Core Slice)
- [ ] **小键盘应用模式 (DECKPAM/DECKPNM)**：支持小键盘状态切换及应答码发送。
- [ ] **字符重复 (REP / CSI Ps b)**：实现重复上一个可打印字符指定次数的功能。
- [ ] **矩形块区域操作**：实现填充（`DECFRA`）、擦除（`DECERA`）、复制（`DECCRA`）、属性修改（`DECCARA`/`DECRARA`）等矩形区域指令。
- [ ] **字符集映射**：支持特殊绘图集（如 `\x1b(0` 线段绘制符号）的正确转换和屏幕渲染。

## 6. 阶段 P4：回写通道与设备查询 (Queries & Replies)
- [ ] **应答指令与报告实现**：基于 PPTY 应答接口实现 DSR (`CSI 6n` 报告游标)、DA1/DA2/DA3、DECRQM/DECRPM（模式查询）与 DECRQSS/XTGETTCAP。
- [ ] **窗口操作命令 (CSI Ps t)**：实现窗口最小化、移动、缩放等安全可控的操作（或按权限限制）。

## 7. 阶段 P5：OSC, DCS 与安全字符串扩展 (Strings & OSC)

## 8. 阶段 P6：交互式输入协议 (Interactive Input)

- [ ] **鼠标追踪支持**：处理 `?9`、`?1000` 等多种鼠标模式，并在 SGR 编码下发送正确的行列与点击松开状态字节。

- [ ] **焦点汇报 (Focus Reporting / ?1004)**：切换窗体焦点时发送对应消息。
